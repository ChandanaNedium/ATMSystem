# ATMSystem

The ATM Interface simulates the functionality of a real ATM machine. Users can perform banking operations such as withdrawing money, depositing money, transferring funds, and viewing transaction history.

//Features

Secure login with user ID and PIN

Withdraw money

Deposit money

Transfer funds

Transaction history

Exit option

//Technologies Used

Java

Scanner class

Menu-driven interface

//How to Run

Compile:

javac ATMSystem.java

Run:

java ATMSystem
